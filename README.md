# Paper

A cli to manage your library account.

## Usage

Currently it only supports listing a session id

`cargo run -- -u A123456789 -p 1234`

or build the executable

```
$ cargo build
...
$ Finished dev [unoptimized + debuginfo] target(s) in 49.65s
$ ./target/debug/paper --username 'A123456789' --password 1234
```
